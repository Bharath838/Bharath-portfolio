## How to Use

1. Clone the repo
2. Install dependencies  
   `npm install`
3. Edit your details in `src/config.json`
4. Add your avatar to `/public/avatar.png`
5. Run locally  
   `npm run dev`
6. Deploy on [Vercel](https://vercel.com/)

## Editing
Change anything in `src/config.json` to instantly update your site!
